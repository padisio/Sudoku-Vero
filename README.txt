# Sudoku PWA — Diagnóstico

1) Sube estos archivos a un repo limpio (raíz).
2) Vercel → New Project → Framework: Other/Static → (sin build) → Deploy.
3) Abre la consola (F12) y valida que aparecen logs:
   - [SW] Registered:
   - [SW] install / activate
4) Comprueba /manifest.webmanifest y /service-worker.js (200 OK).
5) En Android/Chrome: menú ⋮ → Instalar aplicación.
